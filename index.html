Week 7

Things I've Learnt about javascript in Week 7:

  • DOM(Document Object Model)
    • DOM is used to access and modify HTML elements using various methods like selecting the ID, class, or tag name
    • DOM can also be used to respond to users interactions, such as clicks or keyboard input using addEventListener()
  • local storage
    • data can be stored on the user's browser by storing it in their local storage (localStorage.setItem)
    • data can also be retrieved from their local storage (localStorage.getItem)
  • JSON.stringify() converts javascript object into a string
  • JSON.parse() converts a string into a javascript object

    • AJAX allows the communication with a server, exchange data and update the page without requiring the page to reload
  • Postman helps build, test, and modify APIs
